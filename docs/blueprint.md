# **App Name**: Aperture

## Core Features:

- Responsive Grid Gallery: Displays photography in a responsive grid layout that adapts to different screen sizes.
- Click-to-Expand Animation with Description: Enlarges the image upon clicking, dims the background, and reveals the photo's description.
- Individual Project/Detail Pages: Dedicated pages for each photography project, providing deeper storytelling and additional images.
- About + Contact Pages: Essential pages to provide information about the photographer and a contact form.
- SEO Optimization: Implement basic SEO practices to enhance the site's visibility, including optimized titles, descriptions, and alt tags.
- Fast-Loading Images: Ensure images are compressed and lazy-loaded to optimize site loading speed and performance.

## Style Guidelines:

- Background color: Charcoal (#333333) to provide a high-contrast, cinematic backdrop.
- Primary color: Off-white (#F0F0F0) for typography, ensuring readability against the dark background.
- Accent color: Gold (#D4AF37) to highlight key elements and calls to action.
- Headings: 'Inter', a modern sans-serif font, to maintain a clean and contemporary aesthetic.
- Body: 'Lato', a readable sans-serif font, ensuring a smooth reading experience.
- Employ large, edge-to-edge thumbnails and smooth animations for a sleek, immersive user experience.
- Implement a smooth fade-in between pages to enhance user experience.
